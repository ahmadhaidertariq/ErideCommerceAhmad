# Image Assets

Replace these placeholder images with your actual assets:

## Structure

- `placeholder.svg` - Default placeholder (used for case studies, blog posts)
- `hero/` - Hero section images (optional)
- `case-studies/` - Case study cover images
- `blog/` - Blog post featured images
- `services/` - Service icons (or use emoji as in current design)

## Recommended Dimensions

- Hero: 1200x800px or larger
- Case studies: 800x600px (16:10 or 4:3)
- Blog: 800x450px (16:9)
